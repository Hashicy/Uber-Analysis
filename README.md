## 📌 Overview
This repository contains a Tableau visualization project analyzing Uber trip data. The dashboard provides insights into trip patterns, ride volumes, and other key metrics to help understand ride-hailing dynamics. 

Explore the interactive dashboard here: **[Uber Dashboard on Tableau Public](https://public.tableau.com/app/profile/harshita.joshi5538/viz/Uber_17774344572120/Dashboard1?publish=yes)**

## 📂 Dataset
- **File:** `uber_trips_dataset_50k.csv` / `uber_trips_dataset_50k.hyper`
- **Size:** 50,000 trip records
- **Description:** A comprehensive dataset detailing 50,000 Uber trips, enabling deep-dive analyses into time-series trends, location-based demand, and trip characteristics.

## 🛠️ Built With
- **Tableau Desktop / Tableau Public** (`Uber.twb`)
- **Data Source:** CSV & Tableau Hyper Data Extract

## 🚀 Key Insights & Features
- **Temporal Analysis:** Breakdown of trip frequencies by time of day, day of the week, or month.
- **Geospatial Trends:** Identification of high-demand areas and common routes.
- **Interactive Filtering:** Users can dynamically slice the data by various dimensions to uncover specific trends.

## 💻 How to Use Locally
1. Download or clone this repository.
2. Ensure you have [Tableau Desktop](https://www.tableau.com/products/desktop) or [Tableau Reader](https://www.tableau.com/products/reader) installed.
3. Open the `Uber.twb` file to view and interact with the local copy of the dashboard.
4. If prompted, point the data connection to the included `uber_trips_dataset_50k.hyper` or `uber_trips_dataset_50k.csv` file.
